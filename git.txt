1. В личном кабинете на ГитХаб создать новій репозиторий
2. В папке проетка віполнить инициализацию гит
	git init
3. Подключаем удаленній репозиторий
	git remote add НазваниеПроекта "УРЛПутьКПроектуНаГитХабе"
4. Добавить все файлы для отследивания в репозиторий
	git add --all
5. Создать коммит
	git commit -m "Initial commit of the project"
6. Выполнить заливку проекта на удаленный репозиторий
	git push НазваниеПроекта master
